## Getting Started

### Prerequisites
- Java 8 or higher (for Java code)
- GCC/Clang (for C code)
- Python 3.x (for Python scripts)
- A C++ compiler (for C++ code)

### Compilation & Execution

**For Java files:**
```bash
javac FileName.java
java FileName
```

**For C files:**
```bash
gcc -o output filename.c
./output
```

**For Python scripts:**
```bash
python filename.py
```

**For C++ files:**
```bash
g++ -o output filename.cpp
./output
```
